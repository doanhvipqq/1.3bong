# 1.3bong

## Gợi ý nhạc VN (miễn phí / được phép)

Bạn có thể dán URL .mp3 vào ô `URL nhạc` trong panel `BÓNG X` và chọn `Âm VN` hoặc `URL tùy chỉnh`.

- Pixabay (âm thanh miễn phí, tìm 'traditional vietnamese' hoặc 'vietnamese music'):
	- https://cdn.pixabay.com/download/audio/2022/03/15/audio_f95b0adff8.mp3?filename=click-124467.mp3 (ví dụ click)
- FreeSound (kiểm tra giấy phép CC0/CC-BY trước khi dùng):
	- https://freesound.org/ (tìm kiếm 'vietnam' hoặc 'dan tranh')
- Jamendo (âm nhạc độc lập, kiểm tra licence):
	- https://www.jamendo.com

Lưu ý: luôn kiểm tra giấy phép trước khi dùng âm thanh có bản quyền. Tốt nhất là chọn file có giấy phép CCO hoặc rõ ràng cho phép sử dụng miễn phí.
